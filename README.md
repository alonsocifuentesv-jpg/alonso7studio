# L'ÉLITE BARBIER — Landing Page

Página web premium para barbería con sistema de reserva de horas online 24/7.

**Sitio en vivo:** https://alonsocifuentesv-jpg.github.io/alonso7studio/

## Características
- Portada con video/imagen de fondo y llamado a la acción.
- Reserva tipo app: selección de varios servicios, calendario, horarios y pago.
- Pago por transferencia (comprobante por WhatsApp) o pago web.
- Testimonios en carrusel automático.
- Botón de WhatsApp siempre visible.
- 100% responsivo (optimizado para celular).

## Personalización
Todo se edita en un único archivo `index.html`, dentro del bloque
`██ DATOS DEL CLIENTE ██`: nombre del negocio, WhatsApp, servicios, precios,
fotos, video de portada, datos de transferencia, horarios y correos.
